Q&A

1. What is HTML?

HTML stands for Hyper Text Markup Language. It is a language of World Wide Web. It is a standard text formatting language which is used to create and display pages on the Web. It makes the text more interactive and dynamic. It can turn text into images, tables, links



2. What are Tags?

HTML tags are composed of three things: an opening tag, content and ending tag. Some tags are unclosed tag



3. What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and lay out web pages. CSS allows you to control the color, font, spacing, and positioning of elements on a webpage, making it visually appealing and easier to maintain.



4. What is JavaScript?

JavaScript is a programming language used to create dynamic and interactive effects on web pages. It allows you to manipulate the content and behavior of web pages in real-time, such as updating content, validating forms, and handling events.



5. What is a CSS Selector?

A CSS selector is a pattern used to select the elements you want to style. Examples include element selectors (e.g., div), class selectors (e.g., .class-name), and ID selectors (e.g., #id-name). Selectors help apply styles to specific elements in an HTML document.



6. What is a JavaScript Function?

A JavaScript function is a block of code designed to perform a particular task. Functions can take inputs (parameters), execute a series of statements, and return a result. They help in organizing and reusing code efficiently.



7. What is the DOM?

The DOM, or Document Object Model, is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing languages like JavaScript to interact with and manipulate the content, structure, and style of a webpage.



8. What is Responsive Design?

Responsive design is an approach to web design that ensures web pages look good and function well on a variety of devices and screen sizes. It involves using flexible layouts, images, and CSS media queries to adapt the design to different viewports.



9. What is a Bootstrap Framework?

Bootstrap is a popular open-source front-end framework for developing responsive and mobile-first websites. It includes pre-designed HTML, CSS, and JavaScript components that help speed up the design process and ensure consistency across different devices and browsers.



10. What is AJAX?

AJAX, or Asynchronous JavaScript and XML, is a technique used to create interactive web applications. It allows for the asynchronous loading of data from a server without refreshing the entire page, resulting in a smoother and more dynamic user experience.
